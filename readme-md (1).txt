# Motor Unit Pair Analysis

This MATLAB project analyzes motor unit pairs to calculate Î”F and other related metrics. It includes preprocessing, spike train analysis, pair analysis, and visualization tools.

## Getting Started

1. Clone this repository
2. Place your data files in the `data/` directory
3. Run `main.m` in MATLAB

## File Structure

- `src/`: Contains all source code
  - `main.m`: Main script to run the analysis
  - `preprocessing.m`: Data preprocessing functions
  - `spike_train_analysis.m`: Spike train analysis functions
  - `pair_analysis.m`: Motor unit pair analysis functions
  - `plotting.m`: Visualization functions
  - `utils/`: Utility functions
- `tests/`: Contains test scripts
- `data/`: Place your data files here
